# 🚀 Tareas de Programación - CUGDL

¡Hola! Bienvenido a mi repositorio de código. Aquí guardo y organizo las prácticas, ejercicios y proyectos de programación desarrollados durante mi formación académica en el **Centro Universitario de Guadalajara (CUGDL)**.

Este espacio funciona como mi portafolio personal para demostrar mi progreso en la construcción de algoritmos, manejo de lógica estructurada y bases de la computación orientada a datos.

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Entorno de Desarrollo:** Visual Studio Code
* **Control de Versiones:** Git & GitHub

---

## 📂 Estructura del Repositorio

El portafolio está organizado de forma secuencial según la complejidad de los temas abordados durante el semestre:

* **📁 00_Fundamentos_Colecciones:** Primeros pasos con sintaxis básica, variables, manipulación de listas mutables y asignación por referencias en memoria.
* **📁 01_Estructuras_Control:** Implementación de flujos de toma de decisiones utilizando condicionales anidados (`if-elif-else`) y estructuras modernas de selección (`match-case`).
* **📁 02_Operadores_y_Condicionales:** Ejercicios de filtrado lógico de datos y validaciones condicionales dentro de colecciones complejas.
* **📁 03_Librerias_y_Ciclos:** Uso de módulos nativos (como `random`) y control de bucles iterativos y acumulativos para el desarrollo de simulaciones y juegos.
* **📁 04_Funciones_y_Modularidad:** Aplicación del principio de separación de responsabilidades mediante la creación de funciones reutilizables con paso de parámetros y retornos de valor.
* **📁 05_Matrices_y_Graficos:** Manipulación e indexación de arreglos bidimensionales (lista de listas), renderizado de mapas interactivos en consola y graficación básica de funciones.
* **📁 06_Proyectos_Finales:** Aplicaciones integradas en consola que simulan sistemas funcionales (CRUD) empleando diccionarios, persistencia lógica en memoria y control de errores.
* **📁 07_Ciencia_de_Datos_Basica:** Módulos de analítica descriptiva para el cálculo nativo de medidas estadísticas (Media, Mediana, Moda) y algoritmos lógicos para la detección automatizada de anomalías en conjuntos de datos.
* **📁 08_Estructuras_Datos_Avanzadas:** Implementación y simulación desde cero de estructuras de datos lineales abstractas, incluyendo Listas Enlazadas, Pilas (LIFO) y Colas (FIFO) utilizando estructuras nativas.
